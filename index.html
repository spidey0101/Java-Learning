More about Java

What is Java?

Java is a programming language and a platform. Java is a high level, robust, object-oriented and secure programming language. Java was developed by Sun Microsystems (which is now the subsidiary of Oracle) in the year 1995. James Gosling is known as the father of Java.
Before Java, its name was Oak. Since Oak was already a registered company, so James Gosling and his team changed the name from Oak to Java.
Platform: Any hardware or software environment in which a program runs, is known as a platform. Since Java has a runtime environment (JRE) and API, it is called a platform.

Java History

  • 2023: The latest developments and future of Java.
  • 2020: Release of Java 15 with enhanced performance and features.
  • 2018: Java 10 and subsequent versions with incremental updates.
  • 2017: Java 9 introduces the module system.
  • 2014: Release of Java 8 with lambda expressions and the Stream API.
  • 2011: Oracle Corporation acquires Sun Microsystems, becoming the steward of Java.
  • 2004: Release of Java 5 (Java 1.5) with features like generics and metadata annotations.
  • 1996: Introduction of Java Applets for web development.
  • 1995: Java 1.0 released, marking the official launch of Java.
  • 1991: Inception of Java programming language by James Gosling at Sun Microsystems.
